# PI Governance

PI Governance is a local-first governed memory CLI and MCP stdio server for AI agents.

Install:

```bash
cargo install pi-governance-rs
pi --version
```

## Command Matrix

The `pi` CLI includes `init`, `doctor`, `migrate`, `config`, `policy`, `namespace`, `propose`, `review`, `inbox`, `capture`, `memory-worth`, `context`, `session`, `recall-xray`, `graph`, `quality`, `simulate-patch`, `demo`, `agent-instructions`, `apply`, `reinforce`, `supersede`, `tombstone`, `contest`, `resolve-contest`, `retrieve`, `export`, `import`, `list`, `inspect-record`, `list-patches`, `inspect-patch`, `mcp-stdio`, `mcp-config`, `mcp-install`, `mcp-doctor`, `smoke-test`, `release-audit`, and `changelog`.

Documentation: https://github.com/Mont3ll/pi-governance-rs
